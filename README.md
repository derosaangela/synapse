
# Recall every conversation and every connection instantly

Synapse is a second brain for the tech industry that organizes people you meet, companies, open roles, and required skills in a single interface. It serves as a professional memory bank to track every person you meet, their location, and the specific details of your last conversation to ensure no opportunity or connection is forgotten. Creates tailored suggestion messages to reach out. 

This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
